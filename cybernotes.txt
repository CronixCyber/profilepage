Windows Commands
Directory
• cd - Change the current directory
• dir - List files in current directory
• mkdir - Create a directory
• ren - Rename a file


Linux Commands
Directory
• cd - Change the current directory
• ls - List directories
• mkdir - Create a directory
• cp - Copy File
• mv - Move File
• rm - Remove File
• pwd - Display Name of Directory

• cat - Display File Content
• pwd - Display Directory Name
• sudo - Run command as super user
• su - Login as other user
• chmod - File permissions
• chown - File Ownership
• ifconfig - Display Network Card Info
• iwconfig - Display Wireless Network Info
• passwd - Change Password
• shutdown - Shutdown

• ps - Display running processes
• top - Keep displaying processes dynamically
• kill - Modify behavior (remove, restart, pause)
• ls -l | grep host / ls -l | grep file

Tools
• Package Managers - pacman(Arch), dpkg, apt
Commands:
• apt intall - Install Package
• apt Remove - Remove Package
• apt-get update - Update Local Package
• apt-get upgrade - Upgrade all installed Packages


Cisco Configuration
• Ctrl+Shift+6 - Stop Translation
